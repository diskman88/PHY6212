CH6121 BLE SDK概述

CH6121 BLE SDK是基于平头哥低功耗蓝牙芯片CH6121提供的软件开发套件。该开发套件以YoC平台为基础，对蓝牙协议栈做了深度优化和整合，为开发者提供了丰富的蓝牙组件、例程和应用解决方案，涵盖了蓝牙标准Profile、OTA功能以及芯片外设驱动。

CH6121 BLE SDK API

业务服务层 API

  • 基础服务： CLI 命令行服务 | CLI 命令使用说明

系统通用 API

  • 操作系统: 任务 | 互斥锁 | 信号量 | 消息队列 |事件 | 定时器
