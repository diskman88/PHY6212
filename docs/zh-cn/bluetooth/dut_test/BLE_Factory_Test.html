3.RF及功耗工位测试

3.1.框图及流程图

3.1.1.框图

说明：

  • BLE产测集成RF测试和功耗测试。
  • 剑池产测工具通过控制金机和DUT(待测设备)进行RF测试，控制DUT和程控万用表进行功耗测试。
  • 功耗测试需要程控万用表读取DUT电流数据，电流表型号是DM3058。
  • 测量功耗时建议选择稳压电源对DUT进行供电，DM3058程控万用表串联到DUT供电环路。
  • 金机可采用串口单独供电。

3.1.2.流程图

说明：

  • 本工位按照功耗校准，工位配置，RF及功耗测试执行测试
  • RF和功耗测试可以分开执行，不进行功耗测试无需进行功耗校准

3.2.配置

3.2.1.打开剑池产测工具

3.2.2.加载工位配置文件

  • 加载配置文件FACTORY_RF_POWER_V1.1.3

3.2.3.设置

设置界面如下：

3.2.3.1.功耗校准

  • 准备样机

    准备三个产品板并烧录DUT固件（工程目录applications\bluetooth\dut）。

    根据3.1.1所述框图给DUT接线。

    万用表串联到DUT供电环路。

    打开串口工具连接DUT。

    串口输入指令AT+SLEEP让DUT进入休眠。

拔掉串口，从万用表读出当前电流值。

依次读出三个样机的电流值并记录下来。

注意：

电流值需要跟产品板一一对应。

  • 获取DUT串口

    通过计算机设备管理器获取,可通过拔插串口确定。

  • 测量三个样机在sleep模式下的电流，对应写入样机X电流框内

注意：由于Sleep功耗测试对电流较敏感，需要用样机电流进行校准，样机电流单位uA。

  • 获取电源ID

    IO程序库套件安装好后，通过Keysight IO Libraries Suite 的VISA Address项获取

  • 写入电源ID，如USB0::0x2A8D::0x0F02::MY56003881::0::INSTR

  • 执行项选择功耗校准

    用样机校准夹具及串口漏电，只有在手动模式下可以校准。

  • 放入样机1并点击确定

  • 根据提示依次放入样机2和3并点击确定，直到校准完毕

3.2.3.2.其他设置

按如下顺序调整配置：

  • 金机串口

    通过计算机设备管理器获取,可通过拔插串口确定。

  • 频偏上限

    频偏测量高门限，取值为有符号十进制值。单位Khz。

  • 频偏下限

    频偏测量低门限，取值为有符号十进制值。单位Khz。。

  • 线损

    衰减器衰减和射频电缆线损之和，如衰减器设置值88dB,线损是4dB，则该项填写92，需依据实际修改。

  • 功率上限

    DUT发送功率判断门限，取值为有符号十进制值，单位dBm。

  • 功率下限

    同上。

  • 收包上限

    指定衰减下，DUT接收包判别高门限，由于金机最大发包1000包，此处写1000。

  • 收包下限

    DUT接收包判别低门限，默认700。

3.2.3.3.选择执行项

选择如下执行项：

执行项说明：

  • 初始化配置

    DUT和金机配置信息读取，是必选项。

  • 频偏校准

    测量DUT channel 19的频偏值并写入校准值到DUT。

  • TX发送功率和频偏

    测量DUT校准后频偏和发送功率，可选择0，19，38三个channel。

  • 接收灵敏度

    金机发送1000个包，DUT接收包数在门限内通过。可选0，19，38三个channel。

  • 功耗测试

    测量正常工作模式，接收模式和sleep模式下的电流。

  • 卡位

    测试完毕后，等待操作者更换DUT。在自动模式下选择该项，如选择功耗测试可不选卡位(功耗测试自带卡位)。

3.3.执行

  • 开始测试

  • 换板并手动点击开始所有测试进行下次测试

 

3.4.注意事项

  • 耦合测试需要把DUT和金机放到屏蔽箱内。
  • 传导测试如果有多个测试工位，建议测试工位之间至少间隔两米。
  • 如果DUT跟金机之间射频衰减比较大，测试功率可能会有偏差，可以通过小批量测试后，确定发送功率门限值。
