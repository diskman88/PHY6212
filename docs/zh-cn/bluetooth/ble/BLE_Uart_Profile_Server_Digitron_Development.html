Title: BLE数码管开发示例

1.简介

本文是一个数码管显示示例，通过手机APP下发数据，数码管进行显示，数码管选用ZY-2410BW-7做参考。

3.应用开发

该应用主要实现了蓝牙广播、建立连接以及显示数字的功能，包括：

  • 应用初始化
  • 事件处理

3.1.应用初始化

应用入口函数app_main()，主要实现如下功能：

  • 蓝牙协议栈的初始化
  • UART SERVER初始化
  • 开启广播
  • 等待连接并接收数据

3.2.事件处理

3.2.1.接收回调并显示

xxxxxxxxxx
static int uart_server_profile_recv(const uint8_t *data, int length)
{
    g_show_data = 0;
    for(int i = 0; i < length; i++) {
        g_show_data |= data[i] << (i*8);
    }
    LOGD(TAG,"show:%d\r\n",g_show_data);
    digitron_show_off();
    aos_sem_signal(&sync_sem);
    return 0;
}

 

4.例程运行

以CH6121开发板为例，示例程序的运行步骤为：

  • 连接串口调试工具, 配置为 波特率: 115200, 数据位: 8, 校验位: None, 停止位: 1, 流控: None
  • 开发板拨码开关拨至GND
  • 按RESET按键，复位开发板
  • 设备开始广播后，将在串口上输出‘Bluetooth uart server demo’信息
