Title: CLI 命令行服务

概述

CLI(命令行服务)是基于uService的一种服务。可在用户提示符下键入可执行指令，实现对功能命令的维护。

接口定义

初始化命令行服务

xxxxxxxxxx
void cli_service_init(utask_t *task);

根据指定的微服务任务名称和设备名称初始化命令行服务

  • 参数:

    • task: 微服务任务名称

  • 返回值: 无

停止命令行服务

xxxxxxxxxx
int cli_service_stop(void);

命令行服务运行

  • 参数: 无
  • 返回值: 成功返回0，失败返回-1

注册命令行服务

xxxxxxxxxx
int cli_service_reg_.html(const struct cli_command *info);

注册可以执行的命令行服务

  • 参数:

    • info: 需要注册的可以执行命令行服务

  • 返回值: 成功返回0，失败返回负数值

注销命令行服务

xxxxxxxxxx
int cli_service_unreg_.html(const struct cli_command *info);

注销可以执行的命令行服务

  • 参数:

    • info: 需要注销的可以执行命令行服务

  • 返回值: 成功返回0，失败返回负数值

xxxxxxxxxx
int cli_service_reg_.htmls(const struct cli_command commands[],int num_commands);

注册可以执行的命令行服务队列

  • 参数:

    • commands: 需要注册的可以执行命令行服务队列指针
    • num_commands: 需要注册的可以执行命令行服务队列命令数

  • 返回值: 成功返回0，失败返回负数值

xxxxxxxxxx
int cli_service_unreg_.htmls(const struct cli_command commands[], int num_commands);

注销可以执行的命令行服务队列

  • 参数:

    • commands: 需要注销的可以执行命令行服务队列指针
    • num_commands: 需要注销的可以执行命令行服务队列命令数

  • 返回值: 成功返回0，失败返回负数值

xxxxxxxxxx
.html_list_t cli_service_get_.html_list(void);

获取可以执行命令行服务队列指针

  • 参数: 无
  • 返回值: 成功返回服务队列指针，失败返回NULL
